# 色块RPG游戏

从 Egret 引擎重构为纯 HTML/CSS/JavaScript 的放置类RPG游戏。

## 核心玩法

点击不同颜色的色块进行交互，通过完成任务、击败敌人来获取资源，逐步解锁更强的装备和隐藏内容。

## 游戏流程

### 前期（1-10级）
1. 点击蓝色任务块积累金币和经验
2. 升级获得属性点，优先加回复(RPE)
3. 击败小型红色敌人获取钥匙
4. 用钥匙解锁更大的任务和敌人

### 中期（10-30级）
1. 购买ATK/DEF商店物品提升战斗力
2. 解锁特殊商店（生命回复、属性上限）
3. 完成成就解锁宝箱奖励
4. 积累连击数达成最大连击成就

### 后期（30级+）
1. 40级解锁老虎机赌博系统
2. 完成全任务+全敌人解锁Boss
3. 攻击力需200+才能有效伤害Boss
4. 收集全部9只猫解锁最终成就

## 属性系统

| 属性 | 作用 | 恢复 |
|------|------|------|
| LIFE | 生命值 | 自动+RPE加成 |
| ACT | 行动力（做任务消耗） | 自动+RPE加成 |
| ATK | 攻击力 | 自动恢复到上限 |
| DEF | 防御力（上限300） | 自动恢复到上限 |
| RPE | 回复力 | 影响所有恢复速度 |

## 钥匙系统

- **普通钥匙🔑**：击败敌人获得，解锁大型任务/敌人/商店
- **高级钥匙🔑2**：击败敌人获得，解锁特殊功能商店

## 战斗机制

- 伤害公式：`实际伤害 = 敌人攻击 × 0.5 × (1 - DEF/300)`
- 敌人会缓慢回血，需要连续攻击
- 击败敌人后获得金币、经验、钥匙

## 商店机制

- ATK/DEF商店：每个最多购买9次，效果递减
- 生命回复：价格随等级和使用次数增加
- 属性上限：ACT/LIFE最高200

## 成就宝箱

| 成就 | 条件 | 奖励 |
|------|------|------|
| 全任务完成 | 完成所有任务 | 体力上限+20 |
| 全敌讨伐 | 击败所有敌人 | 回复力×2 |
| 全物品购买 | 购买所有商店物品 | 行动上限+20 |
| 最大连击 | 连击达到120 | $200×当前等级 |
| NEKOGAMES | 收集9只猫 | 攻击上限×2 |

## 老虎机（40级解锁）

花费10金币转动，点击停止转轮，三个相同符号获奖：
- $$$ → 1600金币
- @@@ → 200经验  
- 777 → 15540金币
- *** → 9600金币

## 隐藏区域

游戏中间的黑色区域是隐藏内容：
- **老虎机位置**：40级后出现
- **Boss位置**：完成全任务+全敌人后出现
